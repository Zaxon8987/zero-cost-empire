# Zero-Cost Deployment Guide

How to deploy a professional business website to the internet for free.

## Step 1: Create a GitHub Account

Go to https://github.com/signup and create a free account.

## Step 2: Create a Repository

Click the "+" icon > "New repository". Name it `my-business-site`.

## Step 3: Upload Your Files

Upload `index.html` (and any CSS, JS, image files) to the repository.

## Step 4: Enable GitHub Pages

1. Go to your repo > Settings > Pages
2. Under "Branch", select `main` and `/ (root)`
3. Click Save

Your site is live at `https://yourusername.github.io/my-business-site/` within 2 minutes.

## Updating Your Site

Edit any file in the repo and commit. GitHub Pages automatically redeploys.

## Adding a Custom Domain (Optional, Free)

1. Buy a domain from any provider (e.g., .com for ~$10/year, .et domains locally)
2. In your repo Settings > Pages, enter your custom domain
3. Add a CNAME record at your domain provider pointing to `yourusername.github.io`
